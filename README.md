# Edu Tracker

A web application built with **ASP.NET Core** for the backend API and **React** for the frontend UI.

## Requirements

* .NET 10 SDK
* Node.js (LTS version recommended)
* npm

## Getting Started

### Backend Setup

```bash
cd backend\EduTracker
dotnet restore
dotnet run
```

The API will start on the configured port `https://localhost:5270`.

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The React app will run on `http://localhost:3000`.
